# Proyecto-Orga1

# Proyecto Conversor de Sistemas Numéricos - Integrante 1

Este módulo contiene la base del sistema desarrollado en MIPS para la empresa Panita.

## Componentes Incluidos:
- **Macros Obligatorias:** `print_str`, `read_str`, `ascii_to_int`, `print_bit`, `print_nible`, `print_sign`.
- **Interfaz:** Menú interactivo de selección de origen y destino.
- **Captura de datos:** Gestión de buffers para entrada de cadenas de caracteres.
- **Validación:** Soporte para signos (+/-) y detección de punto decimal.
